# Student Record Management System

## Overview
A comprehensive **C-based database management system** designed to efficiently manage student records with core functionalities for adding, updating, deleting, and retrieving student information.

## Features
- ✅ **Add Student Records** - Insert new student entries with validation
- ✅ **Search Functionality** - Query students by ID or name
- ✅ **Update Records** - Modify existing student information
- ✅ **Delete Records** - Remove student entries safely
- ✅ **Display All Records** - View complete student database
- ✅ **File I/O** - Persistent data storage and retrieval

## Technologies & Concepts
- **Language**: C
- **Data Structures**: Arrays, Structs, Dynamic Memory
- **File Handling**: Binary file I/O operations
- **Algorithm Complexity**: Efficient search and sort implementations

## Getting Started

### Prerequisites
- GCC compiler or compatible C compiler
- Linux/Windows terminal or command prompt

### Compilation
```bash
gcc -o student_system cc.c
```

### Running the Program
```bash
./student_system
```

## Usage
The application provides an interactive menu-driven interface:
1. Select operation (Add, Search, Update, Delete, Display, Exit)
2. Follow prompts to enter required information
3. Data persists between sessions via file storage
